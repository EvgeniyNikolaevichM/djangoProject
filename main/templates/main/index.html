WEB-конструктор систем Stewart платформ

Добро пожаловать!

С помощью данного конструктора Вы можете создавать свои Stewart платформы, задавать для них Законы движения и формировать из платформ единые Cистемы Stewart платформ. Управляются платформы (базовые модули системы) на базе сетевого микроконтроллера. Данный WEB-конструктор с распределенной системой SaaS позволяет в режиме онлайн изменять систему (менять количество, расположение и характеристики базовых модулей) для моделирования сложных поверхностей под различные задачи. Конструктор отвечает за хранение, загрузку и выгрузку Законов движения, данных о геометрии Платформ, информацию о сформированных Системах, а объект-представление, основанный на CesiumJS, позволяет пользователю видеть платформы на экране под разными углами, запускать и останавливать анимацию, смотреть текущие координаты. С помощью REST API пользователь может передать софрмированные Законы движения на физические платформы. Зарегистрируйтесь и войдите под своей Учетной записью для того чтобы начать пользоваться конструктором!
